# Tik Tak Toe Game

This is a **tak tak toe** game made using **Python** with **PyGame** for the game structure and Prolog for the game logic, mainly defining the behavior of the game actor that plays the game along with the user.

## Dependencies
- *PyGame >= 2.1.2*
- *PySwip >= 0.2.10*
